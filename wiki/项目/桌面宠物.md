# 桌面宠物

> 像素猫桌宠，PyQt5 + DeepSeek API，已开源
> **GitHub**: https://github.com/Sober05/desktop-pet

## 技术栈

- **GUI**: PyQt5（透明无边框窗口 + QPainter 像素渲染）
- **AI**: DeepSeek API（OpenAI 兼容 SDK）
- **精灵**: 16×16 像素猫，6 种动画状态

## 功能清单

| 功能 | 说明 |
|------|------|
| 桌面漂浮 | 无边框透明窗口，始终置顶，拖拽移动 |
| AI 对话 | 单击猫弹出对话框，接入 DeepSeek 流式回复 |
| 文件分析 | 拖拽文件到猫身上，AI 分析内容并总结 |
| 系统托盘 | 关闭窗口后常驻托盘，双击恢复 |
| 空闲检测 | 5 分钟不操作 → 猫自动睡觉；回来动鼠标 → 醒来问好 |
| 心情系统 | 互动 +3~10 心情，每分钟 -1，低落时主动求关注 |
| 主动搭话 | 根据时间和心情偶尔冒泡聊天 |
| 剪贴板监控 | 复制文字后猫会提示 |
| 快速启动 | 右键菜单快速打开 Obsidian/WPS/VS Code 等 |
| 位置记忆 | 重启后恢复原位 |
| 眼睛跟随 | 光标在猫附近时脸跟着转 |
| 边缘反弹 | 不会被拖出屏幕 |

## 动画状态

| 状态 | 触发条件 |
|------|----------|
| idle | 默认待机，呼吸动画 |
| walk | 自动随机走动 |
| think | 思考中（拖文件分析时） |
| happy | 被点击/摸头/唤醒 |
| sleep | 用户 5 分钟不操作 |
| idle_blink | 偶尔眨眼 |

## 文件位置

`项目/desktop-pet/`

```
desktop-pet/
├── main.py           # 入口 + 崩溃守护 + 开机自启
├── sprites.py        # 16x16 像素猫精灵数据
├── pet_window.py     # 主窗口：托盘/空闲检测/心情/动画/拖放
├── chat_dialog.py    # 对话气泡 UI
├── ai_client.py      # DeepSeek API 封装
├── file_handler.py   # 文件类型识别 + AI 分析 prompt
├── config.json       # API Key 配置 + 位置 + 心情
└── run.bat           # 一键启动脚本
```

## 启动方式

1. 双击 `run.bat`
2. 或桌面快捷方式 `桌面宠物.lnk`

## 注意事项

- 需要 Python 3.14 + PyQt5 + openai
- API Key 在 `config.json` 中配置
- 进程名：python.exe
