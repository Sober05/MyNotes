# 桌面宠物

> SVG 动画猫桌宠，PyQt5 + DeepSeek API + Claude Code 桥接，已开源
> **GitHub**: https://github.com/Sober05/desktop-pet
> **v1.1** — 2026-05-28 安全审计与修复

## 技术栈

- **GUI**: PyQt5（透明无边框窗口 + SVG/像素双模渲染）
- **AI**: DeepSeek API（OpenAI 兼容 SDK）+ Claude Code 桥接（共享目录轮询）
- **精灵**: SVG 动画（46 个文件，clawd/simba 双主题）+ 16×16 像素回退
- **安全**: API Key 环境变量注入（`DEEPSEEK_API_KEY`），单实例 Windows 命名互斥体

## 功能清单

| 功能 | 说明 |
|------|------|
| 桌面漂浮 | 无边框透明窗口，始终置顶，拖拽移动 |
| AI 对话 | 单击猫弹出对话框，接入 DeepSeek 流式回复 |
| 文件分析 | 拖拽文件到猫身上，AI 分析内容并总结 |
| 系统托盘 | 关闭窗口后常驻托盘，双击恢复 |
| 空闲检测 | 5 分钟不操作 → 猫自动睡觉；回来动鼠标 → 醒来问好 |
| 心情系统 | 互动 +3~10 心情，每分钟 -1，低落时主动求关注 |
| 主动搭话 | 根据时间和心情偶尔冒泡聊天 |
| 剪贴板监控 | 复制文字后猫会提示 |
| 快速启动 | 右键菜单快速打开 Obsidian/WPS/VS Code 等 |
| 位置记忆 | 重启后恢复原位 |
| 眼睛跟随 | 光标在猫附近时脸跟着转 |
| 边缘反弹 | 不会被拖出屏幕 |

## 动画状态

| 状态 | 触发条件 |
|------|----------|
| idle_happy | 默认待机 (心情 ≥ 70)：星星眼 + 大笑 |
| idle | 默认待机 (心情 40-69)：正常表情 |
| idle_sad | 默认待机 (心情 < 40)：耷拉耳朵 + 下垂眼 |
| walk | 自动随机走动 |
| think | 思考中（拖文件分析时） |
| happy | 被点击/摸头/唤醒 |
| sleep | 用户 5 分钟不操作 |
| idle_blink | 偶尔眨眼（随心情变体） |

## 文件位置

`项目/desktop-pet/`

```
desktop-pet/
├── main.py            # 入口 + 崩溃守护 + Windows 命名互斥体单实例锁
├── sprites.py         # SVG/像素双模精灵动画 + 主题系统
├── pet_window.py      # 主窗口：渲染/拖放/动画/Claude Code 桥接轮询
├── pet_systems.py     # 后端系统：托盘/菜单/空闲检测/心情/剪贴板/气泡/主动搭话
├── chat_dialog.py     # 对话气泡 UI（流式输出）
├── ai_client.py       # DeepSeek API 封装
├── claude_bridge.py   # Claude Code 共享目录桥接（图片分析）
├── file_handler.py    # 文件类型识别 + AI 分析 prompt
├── config_mgr.py      # 配置管理（env var 优先 → config.json 回退）
├── error_log.py       # 结构化错误日志
├── sound.py           # 音效系统
├── themes.py          # 主题管理（clawd/simba）
├── config.json        # 配置文件（API Key 已迁移至环境变量）
├── chat-history.json  # 聊天历史（最多 50 条）
└── shared/            # Claude Code 桥接临时文件（退出时自动清理）
```

## 启动方式

1. 确保已设置环境变量 `DEEPSEEK_API_KEY`（`setx DEEPSEEK_API_KEY "sk-xxx"`）
2. 双击 `run.bat` 或桌面快捷方式 `桌面宠物.lnk`
3. 或 `python main.py`

## 注意事项

- 需要 Python 3.14 + PyQt5 + openai
- **API Key 不再存放在 config.json**，必须通过 `DEEPSEEK_API_KEY` 环境变量设置
- 进程名：python.exe
- 单实例锁：Windows 命名互斥体（`Global\DesktopPet_SingleInstance_v1`）
