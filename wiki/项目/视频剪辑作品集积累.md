# 视频剪辑作品集积累

> 零经验/作品集空虚时，如何快速积累可展示的剪辑样片。AMV/动漫向优先。

## 核心心态

**不要等"做好"再发，先"有"再说。** 第一个作品大概率不完美，你的目标是"打破零"，不是"一鸣惊人"。

## 最速出片法（1-2 天搞定第一个）

1. **选一首你超熟的歌** — 3 分钟以内，节奏感强
2. **选一部你最熟的番** — 不用到处找素材，脑子里能回忆出画面的
3. **只剪高潮段** — 30-60 秒就够，不用做完整首歌
4. **用剪映/CapCut 就行** — 自动踩点 + 自带转场，1 小时内能出粗剪
5. **加个字幕传 B 站** — 标题直接写「【AMV/新人作】XXX」

## 5 个快速填作品集的策略

### 策略 1：只做 15-30 秒的短片

一首歌的高潮 30 秒，精心剪好，效果远好于勉强的 3 分钟。

> **3 个 30 秒短片 > 1 个 3 分钟长片**

### 策略 2：做"风格练习"系列

每次聚焦一种技法，出一个小样：

| 练习主题 | 产出 | 用时 |
|----------|------|------|
| 卡点踩鼓点 | 《鬼灭之刃 战斗卡点》 | 1 晚 |
| 遮罩转场 | 《你的名字 转场练习》 | 1 晚 |
| 情绪叙事 | 《孤独摇滚 情感练习》 | 1 晚 |
| 调色练习 | 《EVA 赛博朋克风调色》 | 1 晚 |

做 4 个练习 = 马上攒出 4 个能拿出手的作品。

### 策略 3：重剪别人的 BGM，配自己的画面

找一个你觉得"剪得真烂"的热门 AMV，用同一首歌，换你喜欢的番，做你自己的版本。顺便可以发对比视频蹭热度（「我重剪了 XX 的 AMV」）。

### 策略 4：帮圈子里的 UP 主/社团免费剪

漫展宣传片、社团招新片、coser 的 Vlog。零现金收入但收获真实项目经验 + 可以直接放作品集。

### 策略 5：做"教程复盘"视频

一边学一边录：「我学了 XX 转场效果，试做给你看」。发 B 站/小红书，既是作品又是引流内容。

## 作品集包装（3-5 个就够用）

新手不需要整个网站，一个 **B 站首页置顶合集** 或一个 **飞书/Notion 链接** 就够了：

```
📁 作品集（挑最好的 3-5 个放前面）

1. 【AMV】鬼灭之刃 × 残響散歌 — 战斗卡点      (30秒) ★ 最满意的
2. 【MAD】EVA — 刻刻 — 情绪叙事练习            (45秒)
3. 【转场练习】你的名字 — 遮罩转场合集          (30秒)
4. 【调色练习】葬送的芙莉莲 — 胶片风调色        (20秒)
5. 【Vlog】XX 漫展回顾                           (1分钟) ★ 真实项目
```

### 制作标准

- ✅ 分辨率统一 **1080P**，帧率 **30 帧**
- ✅ 字幕字体统一
- ✅ 开头放**最好的片段**，抓住眼球
- ✅ 总时长控制在 **1-2 分钟**
- ✅ 加简单水印保护作品

## AMV/MAD 素材来源

| 资源站 | 用途 |
|--------|------|
| `nyaa.si`（搜日文名） | 下载 BD 生肉 |
| `dmhy.ye1213.com`（动漫花园） | 国内最常用的 BT 资源站 |
| `madsck.com` / `amvscz.com` | MAD/AMV 专用素材库 |
| B 站搜"动漫生肉素材" | 有 UP 主分享剪辑好的片段包 |
| 淘宝/闲鱼搜"AMV 素材包" | 花几块钱买分类好的成品素材 |

> 每次看番顺手截好片段存到「素材库」文件夹，按「打斗/哭泣/特写/风景」分类，一个月就不再缺素材。

## AMV 剪辑技巧速查

### 歌曲结构对应画面逻辑

| 段落 | 画面对应 |
|------|----------|
| 前奏 | 营造气氛（空镜/场景镜头） |
| 主歌 | 引入故事/角色 |
| 副歌 1 | 小高潮 |
| 间奏 | 过渡/情绪积累 |
| 副歌 2 | 爆发高潮 |
| 尾声 | 收尾/留白 |

### 卡点技巧（从低到高）

| 级别 | 方法 |
|------|------|
| 入门 | 按鼓点/重音切镜头 |
| 进阶 | 按音乐波形图找峰值裁剪 |
| 高级 | **情绪同步** — 音乐的叹息 = 角色的眼泪，万马奔腾 = 战场厮杀 |

### 高级转场（别用软件自带的"立方体旋转""翻页"）

| 转场类型 | 做法 |
|----------|------|
| **遮罩转场** | 利用画面元素遮挡镜头时切到下一段（如人物从柱子后走过） |
| **运动匹配** | 上一个镜头物体向左出画 → 下一个镜头物体从右边入画 |
| **同色/同形转场** | 红色的血泊 → 红色的旗帜；月亮 → 圆形钟表 |

### 实操流程

1. **拉片**：逐帧看素材，标记好镜头，分类（打斗/哭泣/特写/爆炸…）
2. **粗剪**：把选好的片段按音乐段落排列到时间轴
3. **精剪**：精细调整卡点和转场
4. **调色**：冷色调=绝望冷酷，暖色调=温馨希望，回忆加褪色效果
5. **导出**：H.264/MP4，码率别太低

## 行动时间线

| 时间 | 目标 |
|------|------|
| **今晚** | 选好歌+番，拉片标记好镜头 |
| **明晚** | 粗剪完成，导出第一版 |
| **这周末** | 发布第一个作品到 B 站 |
| **第 2 周** | 做 2 个风格练习，累积到 3-4 个作品 |
| **第 3-4 周** | 去闲鱼挂商品、在小红书发作品引流 |
| **第 2 个月** | 至少 5 个作品，开始接单 |
