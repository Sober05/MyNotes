# 动漫混剪流水线 (Anime AMV Pipeline)

> Python CLI 自动化动漫 AMV 混剪工具。支持片段分析、BGM 匹配、燃向/情绪向双模板剪辑、AI 标题生成、B站/抖音发布。

## 技术栈

- **Python 3.14** + **MoviePy 2.1** (视频剪辑)
- **librosa** (音频分析/BPM/beat 检测)
- **OpenCV** + **scenedetect** (场景检测)
- **Rich** (CLI 界面)
- **PyYAML** (配置)
- **Claude API** (标题/标签生成)

## 注意事项

- **MoviePy 2.x API 已适配**：subclip→subclipped、volumex→with_volume_scaled、set_audio→with_audio、fx/fl→with_effects/with_updated_frame_function
- **Windows 页文件限制**：编辑器限制最多 12 个片段，最小文件 500KB，最小时长 0.3s，防止 ffmpeg 进程耗尽虚拟内存
- **自动生成质量差**：当前自动剪辑缺乏叙事逻辑，推荐人工剪辑
- **BGM 合成音轨质量低**：建议从 Pixabay 下载真实音乐替换 `bgm/ran/` 和 `bgm/qingxu/` 中的合成 WAV
- **B站登录**：需先 `python main.py login --platform bilibili` 保存 cookie

## 剪辑方案（鬼灭无限列车）

针对 144 个已分析片段，推荐 **炎柱炼狱杏寿郎人物志**，五段式结构：
1. 登场建立 (0-8s)：快节奏短镜头
2. 战斗高潮 (8-35s)：炎柱 vs 上弦叁，卡 BGM 重拍硬切
3. 名台词 (35-50s)：保留原声，BGM 降低
4. 陨落与传承 (50-70s)：情绪爆发+回忆
5. 余韵 (70-90s)：定格微笑收尾

剪辑要点：打斗卡重拍、台词卡空拍、情绪段叠化过渡、关键台词保留原声降低 BGM。
