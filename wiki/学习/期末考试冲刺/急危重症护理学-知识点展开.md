## 🔥🔥 第四章 重症监护（ICU）

---

### 一、GCS — 格拉斯哥昏迷评分（必考🔥）

| 项目 | 评分标准 | 分值 |
|------|----------|------|
| **睁眼反应** | 自动睁开→呼唤睁眼→刺痛睁眼→不睁眼 | 4→3→2→1 |
| **语言反应** | 正常对答→混乱→不当→含糊→无 | 5→4→3→2→1 |
| **运动反应** | 遵命动作→刺痛定位→刺痛回缩→刺痛屈曲→刺痛伸直→无反应 | 6→5→4→3→2→1 |

- 满分 **15分**；**<8分 = 昏迷**；**3分 = 深昏迷**

#### 🔑 名词解释
- **GCS**：格拉斯哥昏迷评分，从睁眼、语言、运动三方面评估意识障碍程度，满分15分，<8分为昏迷

### 二、重要监测指标

| 指标 | 正常值范围 |
|------|------------|
| **CVP**（中心静脉压） | **5~12 cmH₂O** |
| pH | 7.35~7.45 |
| PaO₂ | 80~100 mmHg |
| PaCO₂ | 35~45 mmHg |
| HCO₃⁻ | 22~27 mmol/L |
| BE（碱剩余） | -3~+3 mmol/L |

## 🔥🔥🔥 第五章 心搏骤停与心肺脑复苏（CPCR）— 全书核心

---

### 一、心搏骤停

#### 🔬 三种类型
1. **心室颤动**（VF）— **最常见！首选电除颤**
2. 心脏停搏（心电呈一直线）
3. 无脉性电活动（PEA）

#### 🔑 名词解释
- **心搏骤停**：心脏射血功能突然终止，大动脉（颈动脉、股动脉）搏动消失，重要器官缺血缺氧，如不及时救治将导致死亡
- **心肺脑复苏（CPCR）**：针对心搏骤停的综合急救措施，包括基础生命支持（BLS）、高级心血管生命支持（ACLS）和心搏骤停后治疗

#### BLS顺序
**C→A→B**（胸外按压→开放气道→人工呼吸）

---

### 🔥 CPR核心参数（精准记忆！）

| 参数 | **成人标准** |
|------|------------|
| 按压部位 | 胸骨中下1/3交界处（**两乳头连线中点**） |
| 按压深度 | **5~6 cm** |
| 按压频率 | **100~120 次/分钟** |
| 按压通气比 | **30 : 2**（单人/双人均同） |
| 每次人工呼吸 | ≥1秒，见胸廓起伏 |
| 胸廓回弹 | 保证**完全回弹** |
| 按压中断时间 | 尽量 **<10秒** |

---

### 🔥 电除颤（必考）

| 要点 | 内容 |
|------|------|
| **适应证** | 室颤（VF）+ 无脉性室速（pVT） |
| 双向波除颤仪首次能量 | **200J** |
| 单向波除颤仪 | **360J** |
| 电极板位置 | **心尖部**（左腋前线第5肋间）+ **右锁骨下**（胸骨右缘第2肋间） |
| ⚠️ 除颤时 | **任何人不得接触患者及病床！** |

---

### CPR有效指标

| 指标 | 判断 |
|------|------|
| ✅ 可触及大动脉搏动 | 按压有效 |
| ✅ 瞳孔由大变小 | 脑灌注改善 |
| ✅ 面色由紫绀转为红润 | 氧合改善 |
| ✅ 出现自主呼吸 | 恢复迹象 |
| ❌ 患者挣扎 | **不等于CPR有效** |

---

### 常用药物

| 药物 | 用途 |
|------|------|
| **肾上腺素** | 心搏骤停**首选药物**（1mg q3~5min） |
| 胺碘酮 | 难治性室颤/室速（肾上腺素无效后） |
| 利多卡因 | 室性心律失常（替代胺碘酮） |
| 阿托品 | 症状性心动过缓 |

---

### 二、🔥 有机磷中毒（高频案例分析）

#### 🔬 发病机制
- 有机磷与体内胆碱酯酶结合→胆碱酯酶失活→**乙酰胆碱大量堆积**→胆碱能神经持续兴奋→先兴奋后抑制

#### 🩺 三大症状群

| 症状群 | 表现 |
|--------|------|
| **M样（毒蕈碱样）** | 瞳孔**缩小**（针尖样）、流涎、大汗、呼吸道分泌物↑、肺水肿、心率↓、大小便失禁 |
| **N样（烟碱样）** | **肌纤维颤动**（眼睑、面部、舌→全身）、肌无力 |
| **中枢神经系统** | 头痛、头晕→烦躁、谵妄→昏迷、呼吸中枢抑制 |

**特征性表现：** 呼气有**大蒜味** + 瞳孔缩小 + 大汗

#### 🔑 名词解释
- **阿托品化**：使用阿托品后患者瞳孔较前扩大、口干、皮肤干燥、面色潮红、心率增快、肺啰音减少或消失，提示阿托品用量已达有效剂量

#### 💊 特效解毒剂
1. **阿托品**（抗胆碱能药）：对抗M样症状，原则是**早期、足量、反复**给药
2. **解磷定/氯解磷定**（胆碱酯酶复活剂）：恢复胆碱酯酶活性

#### 🏥 护理诊断
1. **气体交换受损**——与呼吸道分泌物增多、肺水肿有关
2. **清理呼吸道无效**——与分泌物增多、呼吸肌无力有关
3. **潜在并发症**：呼吸衰竭

#### 🤲 护理措施
1. 立即脱离中毒环境，脱去污染衣物（用肥皂水清洗皮肤）
2. 洗胃：2%碳酸氢钠（敌百虫**禁用**！用1:5000高锰酸钾或清水）
3. 阿托品给药至出现**阿托品化**→维持量维持（防阿托品中毒）
4. 保持呼吸道通畅+吸氧
5. 密切观察：瞳孔、呼吸、心率
6. ⚠️ 阿托品中毒表现：瞳孔散大、高热（>40℃）、谵妄、心动过速、尿潴留

## 第七章 常见内外科急症

---

### 一、急性心肌梗死（AMI）

#### 🩺 特征性ECG
- **ST段弓背向上抬高**（损伤型）
- 病理性Q波（坏死型）
- T波倒置（缺血型）

#### 💊 溶栓时间窗：**发病6小时内**（最佳3小时内）

#### 🤲 护理
- 绝对卧床休息；吸氧（急性肺水肿加20%~30%乙醇湿化）
- 镇痛：吗啡（注意呼吸抑制）

### 二、急性脑卒中

| 类型 | 溶栓窗 | 护理要点 |
|------|--------|----------|
| 缺血性 | **4.5小时**内 | rt-PA溶栓 |
| 出血性 | — | **避免搬动**+甘露醇降颅压+控制血压 |

### 三、呼吸衰竭

| 分型 | 血气标准 | 代表疾病 |
|------|----------|----------|
| **Ⅰ型** | PaO₂<60 mmHg | ARDS |
| **Ⅱ型** | PaO₂<60 + PaCO₂>50 mmHg | COPD |

## 🔥 第九章 严重创伤

### ABCDE 初级评估（必考框架🔥）

| 字母 | 内容 | 核心要点 |
|------|------|----------|
| **A** | Airway 气道+颈椎保护 | **最优先！**开放气道同时固定颈椎 |
| **B** | Breathing 呼吸 | 视触叩听评估呼吸；张力性气胸立即减压 |
| **C** | Circulation 循环 | 控制外出血（直接压迫止血）；建立静脉通道 |
| **D** | Disability 神经功能 | 意识水平、瞳孔大小及对光反射、GCS |
| **E** | Exposure 暴露+环境控制 | 充分暴露检查全身后**保暖防低体温** |

### 多发伤急救原则
> 优先处理**危及生命的损伤**：窒息 > 大出血 > 张力性气胸 > 心包填塞

## 第十二章 多器官功能障碍综合征（MODS）

### 🔬 发病机制
- 严重感染/创伤/休克→全身炎症反应失控（SIRS）→多器官功能序贯障碍
- 最常见始动器官：**肺**
- 最常受累器官：**肺**

### 🔑 名词解释
- **MODS**：多器官功能障碍综合征，≥2个器官同时或序贯发生功能障碍的临床综合征
- **SIRS**：全身炎症反应综合征，感染或非感染因素引起的全身性炎症反应

### 🤲 护理
- 控制原发病（感染源控制+抗生素）
- 器官功能支持（呼吸机、CRRT、血管活性药物）

## 第十三章 危重症患者的药物应用与监护

- 血管活性药物：多巴胺、去甲肾上腺素→中心静脉给药，防外渗
- 镇静镇痛：咪达唑仑/丙泊酚→RASS评分评估镇静深度

## 🔥 休克护理总论

### 🔬 五大分类
低血容量性 | 感染性 | 心源性 | 过敏性 | 神经源性

### 🩺 休克体位
**中凹卧位**：头躯干抬高20°~30°+下肢抬高15°~20°

### 🔑 名词解释
- **休克**：有效循环血量急剧减少，组织器官灌注不足，导致细胞代谢紊乱和器官功能障碍的综合征
- **休克指数**：脉率÷收缩压；正常0.5，>2.0=严重休克

### 💊 补液原则
**"先晶后胶、先盐后糖、见尿补钾、宁少勿多"**

### 🏥 护理诊断
1. **心输出量减少**——与有效循环血量不足有关
2. **组织灌注改变**——与微循环障碍有关
3. **潜在并发症**：MODS

### 🤲 护理措施
1. 建立≥2条静脉通道
2. 监测CVP、血压、尿量
3. 过敏性休克→肾上腺素是关键
4. 注意保温但不要过热（过热致血管扩张）

## ⚠️ 易混淆判断题（急危重症篇）

| 命题 | 判断 |
|------|------|
| CPR按压通气比30:2 | ✅ |
| 心搏骤停首选药物是阿托品 | ❌ 首选**肾上腺素** |
| 室颤首选胸外按压 | ❌ 首选**电除颤** |
| 所有中毒患者都应立即洗胃 | ❌ **强酸强碱禁止** |
| 有机磷中毒用阿托品+解磷定 | ✅ |
| 阿托品化表现为瞳孔散大、高热 | ❌ 那是**阿托品中毒** |
| GCS<8分为昏迷 | ✅ |
| 多发伤急救优先处理最明显的骨折 | ❌ 优先处理**危及生命的损伤** |
| 休克补液为先胶后晶 | ❌ **先晶后胶** |
| 溺水患者应常规先控水 | ❌ 现代指南不推荐 |
