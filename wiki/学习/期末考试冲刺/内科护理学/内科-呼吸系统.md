## 一、肺炎 (Pneumonia)

### 一句话定义
由病原微生物、理化因素等引起的终末气道、肺泡和肺间质的急性炎症。

### 病因/诱因
- 细菌最常见（肺炎链球菌占首位）
- 诱因：受凉、劳累、醉酒、上呼吸道感染

### 识别要点（临床表现）
- **典型三联征**：寒战高热（稽留热）+ 咳嗽咳痰 + 胸痛
- 铁锈色痰 = 肺炎链球菌肺炎（大叶性肺炎特征）
- 肺实变体征：触觉语颤增强、叩诊浊音、听诊支气管呼吸音+湿啰音
- 严重者呼吸困难、发绀、感染性休克

### 确诊检查
**胸部 X 线**：肺叶/肺段浸润性实变影（最重要检查）
血常规：WBC↑、中性粒细胞比例↑

### 核心护理诊断
1. 清理呼吸道无效 — 与痰液黏稠、咳嗽无力有关
2. 体温过高 — 与肺部感染有关
3. 气体交换受损 — 与肺实变、通气/血流比例失调有关

### 护理措施
1. **高热护理**：物理降温（冰袋/温水擦浴，禁忌胸腹部酒精擦浴），多饮水 1500-2000ml/d，观察热型
2. **排痰护理**：雾化吸入→拍背（由外向内由下向上）→指导有效咳嗽→体位引流（病变在上痰液重力排出）→必要时吸痰
3. **用药护理**：抗生素（青霉素/头孢）→注意皮试、按时给药、观察疗效和过敏反应
4. **病情观察**：呼吸频率/节律/深度、SpO₂、痰液颜色/量/性状，警惕感染性休克（高热骤降+四肢湿冷+血压下降）
5. **休息**：急性期绝对卧床，胸闷气促者半卧位

### 常考特殊点
- **铁锈色痰 = 肺炎链球菌**
- 抗生素治疗后 3-5 天体温不退→考虑耐药/并发症（脓胸）
- 中毒性肺炎/休克型肺炎→以休克为突出表现+肺部体征不明显

## 二、慢性阻塞性肺疾病 (COPD)

### 一句话定义
以不完全可逆性气流受限为特征的慢性肺部疾病，气流受限呈进行性发展，与肺部对有害气体或颗粒的异常炎症反应有关。

### 病因/诱因
- **吸烟是最重要病因**（占 80-90%）
- 其他：职业粉尘、空气污染、反复呼吸道感染、α₁-抗胰蛋白酶缺乏（遗传因素）

### 识别要点（临床表现）
- **标志性症状**：进行性加重的呼吸困难（最早：劳力性→静息性）
- 慢性咳嗽、咳白色黏痰（晨起明显）
- 视诊桶状胸（肺气肿体征：胸廓前后径增大、肋间隙增宽）
- 叩诊过清音，听诊呼吸音减弱、呼气延长
- 晚期：口唇发绀、杵状指、下肢水肿（肺心病）

### 确诊检查
**肺功能检查是金标准**：吸入支气管舒张剂后 **FEV₁/FVC < 70%**，明确不完全可逆气流受限
FEV₁% 预计值判断严重程度（GOLD 分级）

### 核心护理诊断
1. 气体交换受损 — 与气道阻塞、通气不足有关
2. 清理呼吸道无效 — 与痰液黏稠、呼气肌疲劳有关
3. 活动无耐力 — 与呼吸困难、低氧血症有关

### 护理措施
1. **长期家庭氧疗 LTOT**：低流量 1-2L/min、>15 小时/天→目标 PaO₂≥60mmHg 或 SaO₂≥90%
   - ⚠️ **II 型呼衰患者必须低流量吸氧**（高流量抑制呼吸中枢→加重 CO₂ 潴留）
2. **呼吸功能锻炼**：
   - 缩唇呼吸（鼻吸口呼，吸:呼=1:2~3，嘴唇缩成吹口哨状→增加气道内压防止小气道过早陷闭）
   - 腹式呼吸（鼻吸时腹部隆起→口呼时腹部收缩→手压腹部加压）
3. **有效排痰**：多饮水（>1500ml/d）稀化痰液+雾化+拍背+体位引流
4. **营养支持**：高蛋白高维生素易消化饮食、少食多餐（防腹胀压迫膈肌）
5. **戒烟**是最重要干预

### 常考特殊点
- COPD 气流受限的特点：**不完全可逆**
- 支气管哮喘特点是：**可逆性**气流受限（COPD vs 哮喘经典鉴别点）
- 并发症：自发性气胸（突发胸痛+呼吸困难加剧）、慢性肺源性心脏病、II 型呼吸衰竭

## 三、支气管哮喘 (Bronchial Asthma)

### 一句话定义
由多种细胞和细胞组分参与的气道慢性炎症性疾病，导致气道高反应性和**可逆性**气流受限。

### 病因/诱因
- 本质：**慢性气道炎症**（非细菌感染性炎症）
- 诱因/触发因素：过敏原（尘螨/花粉/动物毛屑）、感染（病毒）、药物（阿司匹林）、运动、情绪激动、冷空气

### 识别要点（临床表现）
- **典型三联征**：发作性喘息 + 胸闷 + 咳嗽（夜间/凌晨加重）
- 呼气性呼吸困难、双肺广泛哮鸣音（呼气相延长）
- 严重发作者（哮喘持续状态）：端坐呼吸、大汗、说话不成句、奇脉、胸腹矛盾运动
- **寂静胸**：极度严重→呼吸音和哮鸣音均减弱→最危险信号

### 确诊检查
- 支气管舒张试验阳性（FEV₁ 增加≥12% 且绝对值增加≥200ml）
- 峰流速仪（PEF）监测变异率

### 核心护理诊断
1. 低效性呼吸型态 — 与支气管痉挛、气道炎症有关
2. 清理呼吸道无效 — 与痰液黏稠、支气管痉挛有关

### 护理措施
1. **急性发作处理**：
   - 脱离过敏原+立即使用速效 β₂ 受体激动剂（沙丁胺醇/特布他林气雾剂）→首选急救药
   - 吸氧（2-4L/min）
   - 指导正确使用定量雾化吸入器 MDI（摇匀→深呼气→含住口含嘴→开始吸气同时按药→深吸气→屏气 10 秒→缓慢呼气）
2. **用药护理**：
   - β₂受体激动剂（沙丁胺醇）：急性发作首选→不良反应心悸手抖
   - 茶碱类（氨茶碱）：有效血浓度与中毒浓度接近→严格遵医嘱，中毒表现恶心呕吐心律失常惊厥
   - 糖皮质激素（布地奈德/氟替卡松吸入→倍氯米松）：控制气道炎症最有效药物，吸入后漱口防念珠菌感染和声嘶
   - 色甘酸钠：预防哮喘发作（运动前/接触过敏原前吸入）
3. **环境管理**：病室清洁干燥、不铺地毯不放花草、避免情绪激动

### 常考特殊点
- 哮喘本质：**慢性炎症**（非感染性）
- 支气管舒张试验→哮喘诊断金标准
- 哮喘 vs COPD 核心鉴别：**可逆性 vs 不完全可逆**
- 夜间频发→提示病情未控制

## 四、肺结核 (Pulmonary Tuberculosis)

### 一句话定义
由结核分枝杆菌引起的慢性呼吸道传染病。

### 病因/传播
- 病原体：结核分枝杆菌（抗酸染色阳性）
- 传染源：排菌的继发性肺结核病人（痰涂片阳性者）
- 传播途径：**飞沫传播**（病人咳嗽/打喷嚏→健康人吸入含菌飞沫）
- 易感因素：免疫功能低下（HIV/糖尿病/使用免疫抑制剂）

### 识别要点（临床表现）
- **全身中毒症状**：午后低热（最常见、最有特征性）、盗汗、乏力、食欲减退、体重减轻
- **呼吸系统症状**：咳嗽咳痰（>2周）、痰中带血/咯血、胸痛
- 体征：肩胛区听诊细湿啰音

### 确诊检查
- **痰结核分枝杆菌检查（涂片/培养）**→确诊最可靠依据
- 胸部 X 线：上叶尖段/下叶背段云絮状浸润灶→纤维增殖→空洞
- PPD 试验（结核菌素试验）→判断是否感染过，非确诊依据：硬结直径（非红晕）≥5mm 阳性

### 核心护理诊断
1. 营养失调：低于机体需要量 — 与慢性消耗代谢增高有关
2. 体温过高 — 与结核菌感染有关
3. 潜在并发症：咯血致窒息

### 护理措施
1. **化疗护理**：
   - 原则：**早期、联合、适量、规律、全程**（十字方针→保证治愈防止耐药）
   - 标准化疗方案：2HRZE/4HR（强化期 2 月：异烟肼 H+利福平 R+吡嗪酰胺 Z+乙胺丁醇 E→巩固期 4 月：异烟肼 H+利福平 R）
   - 全程督导短程化疗 DOTS（医务人员或家属监督服药）
   - 药物不良反应：**异烟肼**→周围神经炎（预防加用 VitB₆）+肝损害；**利福平**→肝损害+尿液/汗液桔红色（告知患者正常现象）；**吡嗪酰胺**→高尿酸血症/关节痛；**乙胺丁醇**→球后视神经炎（视力下降/辨色障碍→出现立即停药查眼科）
2. **咯血护理（高频考点）**：
   - 咯血量：少量<100ml/d、中量 100-500ml/d、大量>500ml/d 或 >300ml/次
   - **咯血窒息是致死主要原因**，先兆：咯血突然减少/中止 + 胸闷烦躁 + 表情恐怖 + 喉头作响→**立即头低足高患侧卧位+轻拍健侧背+清除口鼻血块→吸引器吸痰→高流量吸氧**
   - 咯血时取**患侧卧位**（防止血液流入健侧肺→防窒息和病灶播散）
   - 大咯血时禁食、禁镇咳（可待因等强镇咳药会抑制咳嗽反射→血液无法咳出→窒息）
   - 咯血后漱口清洁口腔
3. **隔离**：痰菌阳性→呼吸道隔离（病室通风+紫外线消毒+病人戴口罩+痰液 20%漂白粉/5%苯酚/焚烧处理）
4. **营养**：高蛋白+高热量+高维生素（结核是慢性消耗性疾病）
5. **健康教育**：坚持全程服药 6-9 个月是关键→不规律服药→耐药→治疗难度大增

### 常考特殊点
- 午后低热+盗汗=结核的经典表现
- 咯血窒息急救：头低脚高患侧卧位（45°-60°）、保持气道通畅
- 咯血时**严禁屏气**（教导病人轻轻将血咳出）
- 肺结核**不通过接触传播**（病人用过的餐具不传染）
- **异烟肼+利福平=均肝损害**→定期查肝功能

## 五、呼吸衰竭 (Respiratory Failure)

### 一句话定义
各种原因引起肺通气和/或换气功能障碍，导致缺氧（PaO₂<60mmHg）伴或不伴二氧化碳潴留（PaCO₂>50mmHg）。

### 病因/诱因
- 气道阻塞（COPD、哮喘、异物）
- 肺实质病变（肺炎、肺水肿）
- 神经肌肉疾病（格林巴利、重症肌无力→呼吸肌麻痹）
- 胸廓畸形

### 识别要点（临床表现）
- **最早症状**：呼吸困难
- **最典型表现**：发绀（缺氧严重时口唇甲床→观察最可靠部位）
- **肺性脑病**（CO₂ 严重潴留→中枢抑制→最严重并发症）：先兴奋（失眠/烦躁/昼夜颠倒）→后抑制（嗜睡→昏睡→昏迷、腱反射减弱消失、扑翼样震颤）
- 循环系统：早期心率增快血压升高→晚期心率减慢血压下降

### 分型与氧疗
|  | I 型呼衰 | II 型呼衰 |
|--|----------|-----------|
| 血气 | PaO₂<60mmHg, PaCO₂ 正常/↓ | PaO₂<60mmHg, PaCO₂>50mmHg |
| 机制 | 换气障碍为主 | 通气障碍为主 |
| 常见病因 | ARDS、肺水肿 | COPD |
| **氧疗** | **高流量 4-6L/min** | **低流量 1-2L/min**（最重要考点！） |
| 原理 | 缺氧无 CO₂ 潴留，高流量快速纠正缺氧 | 高流量会解除低氧对呼吸中枢的兴奋→进一步抑制通气→CO₂ 更高 |

### 核心护理诊断
1. 气体交换受损 — 与肺通气/换气障碍有关
2. 清理呼吸道无效 — 与呼吸肌疲劳、痰液黏稠有关

### 护理措施
1. **保持气道通畅**：吸痰（必要时）+ 雾化 + 拍背 + 有效咳嗽，必要时建立人工气道
2. **合理氧疗**：
   - **I 型呼衰：高流量吸氧 4-6L/min**，目标 PaO₂>60mmHg
   - **II 型呼衰：低流量 1-2L/min**，目标 PaO₂ 60mmHg 左右或 SaO₂ 90%，➡️ **切忌高流量吸氧**
3. **用药护理**：
   - 呼吸兴奋剂（尼可刹米/洛贝林）：用于 II 型呼衰、意识障碍，需要在气道通畅前提下使用→过量→肌肉抽搐/惊厥
   - 禁用镇静催眠药（安定/吗啡）→抑制呼吸中枢
4. **机械通气护理**（无创→有创）：观察人机同步、潮气量、气道峰压、自主排痰能力
5. **病情观察**：血气分析动态监测、意识变化（肺性脑病早期信号）、呼吸频率/节律/深度

### 常考特殊点
- **I 型高流量 → II 型低流量**！这是最重要的考点
- COPD 患者→II 型呼衰→**必须低流量吸氧**
- 发绀 ≠ 缺氧的可靠指标（重度贫血即使缺氧也不发绀）
- 肺性脑病 = II 型呼衰最严重的并发症

## 六、支气管扩张 (Bronchiectasis)

### 一句话定义
支气管及其周围肺组织慢性化脓性炎症，导致支气管壁破坏、管腔持久性扩张和变形。

### 病因/诱因
- 支气管-肺组织感染和支气管阻塞是最主要原因
- 儿童期麻疹/百日咳/支气管肺炎是其重要病因

### 识别要点（临床表现）
- **三大主征**（经典！）：**慢性咳嗽 + 大量脓痰 + 反复咯血**
- 痰液特点：每日痰量可达数百毫升，静置后分三层（上层泡沫→中层混浊黏液→下层脓性坏死组织沉淀）→典型分层现象
- **干性支气管扩张**：仅反复咯血，无明显脓痰

### 确诊检查
- **高分辨率 CT（HRCT）**→确诊金标准，取代支气管造影
- 特征：双轨征/印戒征

### 核心护理诊断
1. 清理呼吸道无效 — 与痰液大量、黏稠有关
2. 潜在并发症：大咯血窒息

### 护理措施
1. **体位引流（最重要护理措施）**：
   - 病变部位在上方→支气管开口向下→引流液靠重力排出
   - 每次 15-20 分钟，每日 2-3 次，**饭前 1h 或饭后 1-2h 进行**（空腹或餐后充分消化→防止呕吐误吸）
   - 引流中观察：出现咯血/头晕/发绀/呼吸困难→立即停止
   - 引流后漱口清洁口腔
2. **促进排痰**：多饮水稀化痰液+雾化吸入+拍背震颤
3. **咯血护理**：参见肺结核咯血护理

### 常考特殊点
- 痰液**三层分层现象**→高度提示支气管扩张
- 咯血**患侧卧位**
- 体位引流安排在**饭前**（非饭后！）
- 体位引流病变部位在**上**

## 本章小结：高频数字与对比

| 考点 | 关键数字/对比 |
|------|---------------|
| COPD 肺功能诊断 | FEV₁/FVC < 70% |
| COPD 氧疗流量 | 低流量 1-2 L/min，>15h/d |
| II 型呼衰氧疗 | **低流量 1-2 L/min** |
| I 型呼衰氧疗 | **高流量 4-6 L/min** |
| 哮喘本质 vs COPD | 可逆性气流受限 vs 不完全可逆 |
| 咯血量分级 | 少量<100ml/d、中量 100-500ml/d、大量>500ml/d |
| 结核化疗原则 | 早期+联合+适量+规律+全程 |
| 结核化疗时长 | 6-9 个月 |
| 支扩痰液特点 | 三层分层现象 |
| 大咯血体位 | **患侧卧位**，头低足高 45-60° |
