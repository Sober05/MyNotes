## 一、急性肾小球肾炎 (Acute Glomerulonephritis)

### 一句话定义
A 组 β 溶血性链球菌感染后诱发的免疫性肾小球炎症。

### 病因
- **前驱感染**：链球菌感染（扁桃体炎/咽炎/皮肤感染）1-3 周后发病（抗原抗体免疫复合物沉积→肾小球）
- 好发：**儿童/青少年**（学龄期多见）

### 识别要点
- 典型三联征：**血尿 + 水肿 + 高血压**
- 血尿：几乎 100% 有（肉眼血尿→洗肉水色或浓茶色；镜下血尿）
- 水肿：最常见的首发症状，**晨起眼睑/颜面部水肿**→下行性
- 高血压：一过性，轻-中度
- 可伴一过性肾功能损害（少尿）

### 确诊/辅助检查
- **尿液检查**：镜下血尿（变形红细胞）+ 蛋白尿（轻-中度）+ 管型尿（红细胞管型）
- **ASO（抗链球菌溶血素 O 抗体）↑**：提示近期链球菌感染
- 补体 C3 ↓（8 周内恢复正常，不恢复→考虑其他类型肾炎）
- 肾功能一过性异常

### 核心护理诊断
1. 体液过多 — 与肾小球滤过率下降、水钠潴留有关
2. 潜在并发症：严重循环充血、高血压脑病、急性肾衰竭

### 护理措施
1. **休息最重要**：急性期绝对卧床休息 2 周→症状消失+尿检好转后逐步增加活动量
2. **饮食护理**：
   - **低盐饮食**（<3g/d）：控制水肿和高血压
   - **限水**：水肿重者量入为出（前日尿量+500ml）
   - 氮质血症期→**低蛋白饮食**（优质蛋白 0.5g/kg·d），肾功能恢复后正常蛋白
3. **病情观察**：
   - 准确记录 24h 出入量、每日测体重
   - 监测血压 q4-6h
   - 注意尿量（少尿→无尿，警惕急性肾衰竭）
   - 警惕严重循环充血（呼吸困难+肺部湿啰音+心率快+肝大）
   - 警惕高血压脑病（剧烈头痛+呕吐+视力模糊+惊厥）
4. **用药护理**：
   - 利尿剂（呋塞米/氢氯噻嗪）→监测电解质（低钾）
   - 降压药（硝苯地平/卡托普利）
5. **预防感染**：注意保暖、口腔卫生、皮肤清洁

### 常考特殊点
- 前驱链球菌感染 1-3 周后发病（不是立即！）
- 水肿从**眼睑**开始（肾脏病性水肿 vs 心源性从下肢开始）
- 几乎 100% 有**血尿**
- 本病为**自限性**疾病→预后好、多数可痊愈

## 二、肾病综合征 (Nephrotic Syndrome)

### 一句话定义
由多种病因引起肾小球基底膜通透性增高，导致大量蛋白尿、低蛋白血症、水肿和高脂血症的一组临床综合征。

### 病因
- 原发性（大部分）：微小病变型肾病（儿童最常见）、膜性肾病（成人最常见）
- 继发性：SLE、糖尿病肾病、过敏性紫癜、多发性骨髓瘤

### 识别要点
**四大特征（诊断标准）：**
1. **大量蛋白尿** > 3.5g/d（核心特征！最基本病变）
2. **低蛋白血症**：血浆白蛋白 < 30g/L
3. **水肿**：全身性凹陷性水肿，**最早出现且最突出**→眼睑→下肢→全身
4. **高脂血症**：胆固醇↑、甘油三酯↑

### 确诊/辅助检查
- **24 小时尿蛋白定量 > 3.5g**（最重要诊断依据）
- 血浆白蛋白 < 30g/L
- 肾穿刺活检（确定病理类型）

### 核心护理诊断
1. 体液过多 — 与大量蛋白尿致低蛋白血症有关
2. 营养失调：低于机体需要量 — 与蛋白丢失有关
3. 有感染的危险 — 与使用免疫抑制剂、低蛋白血症有关

### 护理措施
1. **休息**：严重水肿和高血压时卧床→腹水者半卧位→症状缓解后适当活动
2. **饮食护理**：
   - 正常量优质蛋白（1.0g/kg·d）**不是高蛋白饮食！**（高蛋白增加肾脏负担+加重蛋白尿）
   - 低盐（<3g/d）+ 限水
   - 低脂饮食（控制高脂血症）
3. **皮肤护理（重点！）**：
   - 高度水肿→皮肤菲薄易破损→床铺整洁干燥、勤翻身、穿宽松柔软衣裤
   - 阴囊水肿者→丁字带托起
   - 避免肌肉注射→药物吸收差+易形成皮下脓肿
4. **用药护理**：
   - **糖皮质激素（泼尼松）为首选治疗**→起始足量（1mg/kg·d 8-12 周）→缓慢减量（每 2-3 周减原量 5-10%）→长期小剂量维持
   - 利尿剂→监测电解质和血容量
   - 环磷酰胺等免疫抑制剂→骨髓抑制/肝损害/出血性膀胱炎（多饮水+排空膀胱）
   - **ACEI/ARB**→降蛋白尿+保护肾功能
5. **预防感染**：感染是最常见并发症和复发诱因→病室清洁通风，限制探视，注意保暖和口腔卫生

### 常考特殊点
- **大量蛋白尿是最基本特征**（不是水肿！）
- 饮食是**正常量**优质蛋白，不是高蛋白！
- **感染**是最常见并发症
- 激素治疗原则：起始足量→缓慢减量→小剂量维持（不可突然停药→肾上腺皮质功能不全）
- 肾静脉血栓是重要并发症（高凝状态→利尿过度+长期激素→防血栓）

## 三、慢性肾衰竭 (Chronic Renal Failure, CRF)

### 一句话定义
各种慢性肾脏疾病进行性发展，导致肾单位和肾功能不可逆性丧失，最终形成以代谢产物潴留、水电解质和酸碱平衡紊乱及多系统受累为表现的临床综合征。

### 病因
- 我国最常见原因：原发性肾小球肾炎
- 其次：糖尿病肾病（近年来快速增加）、高血压肾损害
- 其他：多囊肾、梗阻性肾病

### 临床分期

| 分期 | GFR (ml/min) | 特点 |
|------|-------------|------|
| 代偿期 | 80-50 | 无症状或原发病表现 |
| 失代偿期（氮质血症期） | 50-25 | 乏力+食欲减退+夜尿增多+轻度贫血 |
| 肾衰竭期（尿毒症早期） | 25-10 | 消化道症状明显+贫血+水电解质紊乱 |
| 尿毒症期（ESRD终末期） | <10 | 全身多系统受累→需肾脏替代治疗 |

### 多系统临床表现（按重要性排序）

| 系统 | 表现 | 考点 |
|------|------|------|
| **消化系统** | **最早和最突出表现**：厌食、恶心、呕吐、食欲不振、口臭（尿味）、消化道出血 | 为什么最早？→尿素从消化道排出→细菌分解产氨→刺激胃肠黏膜 |
| **心血管系统** | 高血压（容量依赖性+肾素依赖性）；尿毒症性心包炎（心包摩擦音）；心衰→**心血管疾病是主要死因** | — |
| **血液系统** | **肾性贫血**：EPO 缺乏→正细胞正色素性贫血（面色苍白、乏力、心悸）；出血倾向（血小板功能↓→皮肤瘀斑、鼻衄、消化道出血） | 补充重组人促红素 rHuEPO（监测 Hb 目标 110-120g/L）+ 补铁（EPO 需要铁原料） |
| **水电解质/酸碱** | **高钾血症最危险**（肾排钾↓+代谢性酸中毒使细胞内钾移出→可致心脏骤停）；代谢性酸中毒（深大呼吸）；低钙高磷（肾活化 VitD↓→钙吸收↓→肾性骨营养不良/肾性佝偻病→骨质疏松、骨痛、病理性骨折）；水肿或少尿 | 高钾处理（见外科水电酸中毒章节） |
| **神经肌肉** | 尿毒症脑病（注意力不集中→嗜睡→昏迷）；周围神经病变（下肢远端对称性感觉异常→"不宁腿"） | — |
| **皮肤** | 皮肤干燥脱屑+**尿素霜**（尿毒症面容：面部肤色深暗、轻度浮肿）；**皮肤瘙痒**最常见 | — |

### 确诊与分期检查
- **血肌酐 Scr ↑**、**尿素氮 BUN ↑**、**GFR ↓**（分期的金标准）
- 血电解质（警惕高钾）、血气分析（代酸）
- 肾 B 超（双肾缩小→提示慢性病变）

### 核心护理诊断
1. 营养失调：低于机体需要量 — 与消化道症状、饮食限制有关
2. 体液过多 — 与肾小球滤过↓、水钠潴留有关
3. 潜在并发症：高钾血症、急性左心衰、肾性骨病

### 护理措施

**1. 饮食护理（高频考点！）：**
| 营养素 | 原则 | 具体 |
|--------|------|------|
| 蛋白质 | **优质低蛋白** | 0.6-0.8g/kg·d，其中优质蛋白占 50-70%（鱼/瘦肉/蛋/奶） |
| 热量 | **高热量** | 30-35kcal/kg·d（糖+脂肪→减少蛋白分解产尿素） |
| 钠 | 低盐 | 3g/d（有水肿高血压） |
| 钾 | **低钾** | 少食香蕉/橙子/土豆/蘑菇/深色蔬菜（先焯水去钾） |
| 磷 | **低磷** | 少食动物内脏/坚果/全麦（高磷加重肾性骨病） |
| 水 | 量入为出 | 不显性失水 500-600ml+前日尿量 |

**2. 病情观察：**
- 称体重每日 1 次（同一时间、同秤、同衣着）
- 记 24h 出入量
- **监测高钾血症表现**（口周/四肢麻木→心率减慢→心律不齐→最危险→心脏骤停于舒张期→**T 波高尖是早期心电图表现**）
- 监测血压、呼吸、意识

**3. 高钾血症急救护理：**
- 立即停含钾食物药物
- 10% 葡萄糖酸钙 iv（拮抗钾对心脏毒性→保护心肌，最快起效）
- 50% 葡萄糖+胰岛素 iv→将 K⁺ 转移入细胞内
- 碳酸氢钠→纠正酸中毒+促 K⁺ 入胞
- 透析（最有效方法）

**4. 用药护理：**
- 重组人促红素（rHuEPO）→皮下注射→**Hb 目标 110-120g/L，不宜过高**（>130 增加心血管风险）→需补铁+叶酸
- 降压药（ACEI/ARB 首选→降蛋白尿+延缓肾衰进展，但注意血钾↑和 Scr↑）
- 磷结合剂（碳酸钙）→**餐中嚼服**降磷
- 活性维生素 D（骨化三醇）→防肾性骨病
- 避免使用肾毒性药物（氨基糖苷类抗生素/NSAID/造影剂）

**5. 透析护理（血液透析 vs 腹膜透析）：**
- 血液透析（HD）→动静脉瘘（AVF）护理：
  - **禁止在造瘘侧肢体测血压、抽血、输液、提重物！**
  - 每日检查瘘口震颤和杂音（震颤消失→瘘口血栓/堵塞→立即报告医生）
  - 抬高造瘘侧肢体、握拳运动促进瘘口成熟
  - 透析前后称体重
- 腹膜透析（PD）→严格无菌操作+引流液观察+防腹膜炎

**6. 皮肤护理：**
- 皮肤瘙痒→温水擦洗+涂润肤剂（勿用肥皂酒精→加重干燥）
- 剪短指甲→防抓破皮肤
- 尿素霜→涂皮肤

### 常考特殊点
- **最早出现**：消化道症状（食欲不振）
- **主要死因**：心血管疾病
- **最危险**：高钾血症→心脏骤停
- 动静脉瘘：**禁止**测血压/抽血/输液/提重物！
- 优质**低**蛋白 + **高**热量饮食
- **低磷**饮食（高磷→加重肾性骨病）
- Hb 目标 110-120g/L（不要超过 130）
- T 波高尖 = 高钾的早期心电图表现

## 本章小结：泌尿系统护理数字要点

| 考点 | 数字/关键 |
|------|----------|
| 急性肾炎前驱感染间隔 | 1-3 周 |
| 急性肾炎急性期卧床 | 2 周 |
| 肾病综合征尿蛋白 | > 3.5g/24h |
| 肾病综合征血浆白蛋白 | < 30g/L |
| 肾病综合征饮食蛋白 | 正常量优质蛋白 1.0g/kg·d（非高蛋白！） |
| CRF 饮食蛋白 | 优质低蛋白 0.6-0.8g/kg·d |
| CRF 热量 | 高热量 30-35kcal/kg·d |
| rHuEPO 目标 Hb | 110-120g/L |
| CRF 饮水 | 前日尿量 + 500ml |
| 急性肾炎水肿部位 | 眼睑→下行性（vs 心源性：下肢→上行性） |
| 肾病综合征最常见并发症 | 感染 |
| CRF 主要死因 | 心血管疾病 |
| 高钾最危险 | 心脏骤停于舒张期 |
| 动静脉瘘护理核心 | 禁止测血压/抽血/输液 |
| 高磷血症对策 | 磷结合剂**餐中嚼服** |
