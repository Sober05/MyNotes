## 一、肾小球疾病概述

### 🔑 名词解释

| 名词 | 定义 |
|------|------|
| **肾小球疾病** | 一组以肾小球损害为主的疾病，主要表现为蛋白尿、血尿、水肿、高血压及肾功能损害 |
| **肾源性水肿** | 肾脏疾病导致的水肿，因大量蛋白尿致低蛋白血症（胶体渗透压↓）或肾小球滤过率↓致水钠潴留 |
| **膀胱刺激征** | 尿频、尿急、尿痛三联征，提示膀胱或尿道受炎症、机械或化学刺激 |
| **蛋白尿** | 尿蛋白定量>150mg/24h；大量蛋白尿指>3.5g/24h |
| **血尿** | 离心尿沉渣镜检RBC>3个/HP |
| **白细胞尿（脓尿）** | 离心尿沉渣镜检WBC>5个/HP |
| **少尿** | 24h尿量<400ml（或<17ml/h） |
| **无尿** | 24h尿量<100ml |
| **多尿** | 24h尿量>2500ml |
| **夜尿增多** | 夜间尿量>750ml或超过白天尿量 |
| **管型尿** | 尿中出现由蛋白质、细胞或其碎片在肾小管内凝固形成的圆柱状物；**蜡样管型**对慢性肾衰竭有诊断意义 |

### 📋 常见症状体征的护理诊断与措施

**肾源性水肿：**
- 护理诊断：体液过多——与水钠潴留、低蛋白血症有关
- 护理措施：①限盐<3g/d；②记录24h出入量+每日测体重；③严重水肿卧床抬高下肢；④皮肤护理防压疮

**膀胱刺激征：**
- 护理诊断：排尿障碍——与尿路感染所致膀胱刺激有关
- 护理措施：①多饮水>2000ml/d；②勤排尿、不憋尿；③保持会阴清洁；④遵医嘱抗生素治疗

## 二、急性肾小球肾炎（AGN）

### 🔬 发病机制
- β-溶血性链球菌"A族致肾炎菌株"感染后，通过**免疫复合物沉积**（循环免疫复合物或原位免疫复合物）激活补体系统，导致肾小球弥漫性增生性炎症
- 感染后**1~3周**潜伏期（上呼吸道感染后约1~2周，皮肤感染后约2~3周）

### 🩺 临床表现
1. **血尿**：首发症状，几乎100%出现，30%为肉眼血尿
2. **水肿**：晨起眼睑颜面水肿（最典型），可波及全身
3. **高血压**：一过性轻中度升高，与水钠潴留有关
4. 可伴一过性肾功能减退、尿量减少

### 🔑 名词解释
- **急性肾小球肾炎**：β-溶血性链球菌感染后引起的免疫复合物性肾小球肾炎，以血尿、水肿、高血压和蛋白尿为主要表现的**自限性疾病**

### 💊 治疗要点
1. **本病为自限性疾病，以对症治疗为主**
2. 休息：急性期卧床2~3周，直至肉眼血尿消失、水肿消退、血压正常
3. 限盐限水：有水肿高血压时限盐<3g/d
4. 利尿：氢氯噻嗪
5. 降压：硝苯地平
6. 抗生素：青霉素清除链球菌感染灶（10~14天）

### 🏥 护理诊断
1. **体液过多**——与肾小球滤过率↓致水钠潴留有关
2. **活动无耐力**——与水肿、高血压有关
3. **潜在并发症**：急性肾损伤、高血压脑病

### 🤲 护理措施
1. **休息**：急性期绝对卧床2~3周（减少代谢产物，减轻肾负担），待肉眼血尿消失、水肿消退、血压正常后可下床活动，血沉正常可上学，Addis计数正常可恢复体力活动
2. **饮食**：限盐<3g/d；限水（前日出量+500ml）；氮质血症时限蛋白0.5g/kg/d；高糖提供能量
3. **病情观察**：每日测血压、记出入量、测体重；观察有无头痛呕吐（高血压脑病）
4. **用药护理**：利尿剂观察电解质；降压药防直立性低血压
5. **健康教育**：预防上呼吸道和皮肤感染；感染后2~3周查尿常规

## 三、慢性肾小球肾炎（CGN）

### 🔬 发病机制
- 起始因素多为**免疫介导炎症**，非免疫因素（高血压、大量蛋白尿、高脂血症等）参与疾病进展
- 双侧肾脏弥漫性肾小球硬化、肾小管萎缩、间质纤维化
- 病程>1年，最终发展为慢性肾衰竭

### 🩺 临床表现
1. **蛋白尿**：必备表现，多为轻中度
2. **血尿**：镜下血尿多见
3. **高血压**：持续性中等以上升高
4. **水肿**：晨起眼睑面部，或下肢凹陷性
5. 肾功能进行性减退

### 🔑 名词解释
- **慢性肾小球肾炎**：以蛋白尿、血尿、水肿、高血压为基本临床表现，起病方式各异、病情迁延、缓慢进展，终将发展为慢性肾衰竭的一组肾小球疾病

### 💊 治疗要点
- 目标：**延缓肾功能恶化**，非消除蛋白尿
1. **控制高血压**：首选**ACEI/ARB**（卡托普利/缬沙坦）—既降压又减少蛋白尿、延缓肾功能恶化
2. 限盐<3g/d；**优质低蛋白饮食**（0.6~0.8g/kg/d）
3. 抗血小板聚集：双嘧达莫

### 🏥 护理诊断
1. **体液过多**——与肾小球滤过率↓有关
2. **营养失调：低于机体需要量**——与限制蛋白饮食有关
3. **焦虑**——与疾病迁延、预后不良有关
4. **潜在并发症**：慢性肾衰竭

### 🤲 护理措施
1. **饮食**：优质低蛋白（0.6~0.8g/kg/d，动物蛋白为主）+高热量+低盐+低磷
2. **用药**：ACEI/ARB注意血钾升高；降压药长期规律服勿骤停
3. **避免加重因素**：防感染、劳累、脱水、使用肾毒性药物（氨基糖苷类）
4. **定期复查**：尿常规、肾功能、血压

## 四、🔥 肾病综合征（NS）

### 🔬 发病机制
- **肾小球滤过膜通透性↑↑**（足细胞损伤、基底膜病变）→**大量蛋白尿**（核心环节）
- 大量蛋白尿→肝脏代偿性合成白蛋白↑+蛋白从尿丢失→**低蛋白血症**（血浆胶体渗透压↓）
- 胶体渗透压↓→水分从血管进入组织间隙→**水肿**
- 肝脏合成蛋白↑的同时脂蛋白合成也↑→**高脂血症**

### 🩺 临床表现
**四大特征：**
1. **大量蛋白尿**：>3.5g/24h（诊断必备）
2. **低蛋白血症**：血浆白蛋白<30g/L
3. **水肿**：全身凹陷性，眼睑→下肢→全身→浆膜腔积液
4. **高脂血症**：胆固醇↑、甘油三酯↑、LDL↑

**并发症：**
- ①**感染**（最常见！呼吸道、泌尿道、皮肤）
- ②**血栓栓塞**（肾静脉血栓最常见，高凝状态+脱水→危险）
- ③急性肾损伤
- ④蛋白质及脂代谢紊乱

### 🔑 名词解释
- **肾病综合征**：由各种原因所致肾小球滤过膜通透性增高，以大量蛋白尿（>3.5g/24h）、低蛋白血症（<30g/L）、高度水肿、高脂血症为特征的临床综合征
- **大量蛋白尿**：24小时尿蛋白定量>3.5g

### 💊 治疗要点
1. **糖皮质激素（首选）**：泼尼松，原则：**起始足量、缓慢减量、长期维持**（半年至1年以上）
2. **利尿消肿**：氢氯噻嗪、呋塞米（注意电解质）
3. **免疫抑制剂**：环磷酰胺（激素依赖或抵抗时），注意骨髓抑制、出血性膀胱炎
4. ACEI/ARB：减少蛋白尿
5. 抗凝：预防血栓（低分子肝素）

### 🏥 护理诊断
1. **体液过多**——与低蛋白血症致胶体渗透压下降有关
2. **营养失调：低于机体需要量**——与大量蛋白丢失有关
3. **有皮肤完整性受损的危险**——与高度水肿有关
4. **有感染的危险**——与免疫力下降、使用激素/免疫抑制剂有关
5. **潜在并发症**：血栓栓塞、急性肾损伤

### 🤲 护理措施
1. **休息**：严重水肿卧床、抬高下肢；缓解后可适当活动
2. **饮食**：正常量优质蛋白（0.8~1.0g/kg/d，过高加重肾负担）；低盐<3g/d；限水；低脂
3. **皮肤护理**：保持清洁干燥、床铺平整、定时翻身、注射用5~6号细针头、拔针后延长按压
4. **用药护理**：激素不可擅自停药/减量（肾上腺皮质危象）；观察感染、高血糖、高血压、骨质疏松、消化道溃疡等副作用；免疫抑制剂多饮水+定期血象
5. **预防感染**：病室通风消毒、限制探视、口腔护理
6. **病情观察**：每日测体重、记出入量、监测尿蛋白；注意有无胸痛、咯血（肺栓塞）或单侧肢体肿胀（深静脉血栓）

## 五、尿路感染（UTI）

### 🔬 发病机制
- **上行感染**（最主要途径）：细菌从尿道口→膀胱→输尿管→肾盂
- 最常见致病菌：**大肠杆菌**（革兰阴性杆菌，占70~90%）
- 易感因素：女性尿道短直、性生活、妊娠、尿路梗阻、导尿、糖尿病

### 🩺 临床表现
| | 膀胱炎 | 急性肾盂肾炎 |
|------|--------|------|
| 膀胱刺激征 | +++ | + |
| 发热寒战 | — | +++ |
| 肾区叩击痛 | — | +++ |
| 全身症状 | 不明显 | 明显（头痛、恶心） |
| WBC尿 | + | +（可有WBC管型） |

### 🔑 名词解释
- **尿路感染**：病原体在尿路中生长繁殖引起的感染性炎症，分为上尿路感染（肾盂肾炎）和下尿路感染（膀胱炎）
- **膀胱刺激征**：尿频、尿急、尿痛，提示膀胱或尿道受刺激（炎症最常见）

### 💊 治疗要点
1. 抗生素：喹诺酮类（左氧氟沙星）、头孢类、SMZco
2. 膀胱炎：3天短程疗法
3. 急性肾盂肾炎：抗生素2周
4. 碱化尿液：碳酸氢钠（减轻膀胱刺激征）

### 🏥 护理诊断
1. **排尿障碍**——与尿路感染所致膀胱刺激征有关
2. **体温过高**——与感染有关

### 🤲 护理措施
1. **多饮水**：>2000ml/d（冲刷尿路）→尿量>1500ml/d
2. **勤排尿**：每2~3h排尿一次，勿憋尿
3. **个人卫生**：便后从前向后擦拭；穿棉质内裤；经期勤换卫生巾
4. **用药护理**：抗生素全程规律、勿自行停药；磺胺类多饮水防结晶
5. **健康教育**：性生活后立即排尿；避免盆浴；复查尿常规

## 六、急性肾损伤（AKI）

### 🔬 发病机制
- **肾前性**：有效循环血量↓→肾灌注↓（休克、脱水）
- **肾性**：肾实质损伤（急性肾小管坏死ATN最常见）
- **肾后性**：尿路梗阻

### 🩺 临床表现（三期）
| 分期 | 特点 |
|------|------|
| **少尿期** | 少尿/无尿（7~14天）+高钾+高镁+高磷+低钙+水中毒+代谢性酸中毒+氮质血症 |
| **多尿期** | 尿量渐↑（可达3000~5000ml/d），注意脱水+低钾 |
| **恢复期** | 肾功能恢复（3~12月） |

### 🏥 护理诊断
1. **体液过多**——与肾小球滤过率↓有关（少尿期）
2. **有体液不足的危险**——与多尿有关（多尿期）
3. **潜在并发症**：高钾血症、代谢性酸中毒

### 🤲 护理措施
1. **少尿期**：严格控制液体入量="量出为入"（前日出量+500ml）；限钾（禁香蕉橘子）；限蛋白；高热量
2. **多尿期**：补液补电解质；预防脱水
3. **透析**：严重高钾/酸中毒/液体超负荷时紧急透析

## 七、🔥 慢性肾衰竭（CRF）

### 🔬 发病机制
- 各种慢性肾病进行性破坏肾单位→**健存肾单位进行性减少**→肾小球高滤过（"三高"：高压+高灌注+高滤过）→残余肾单位硬化→肾功能不可逆丧失
- 最终进入**尿毒症期**
- 常见原发病：慢性肾炎（我国最常见）、糖尿病肾病、高血压肾病

### 🩺 临床表现（全身各系统受累）

| 系统 | 表现 |
|------|------|
| **消化系统** | **最早最突出**：食欲不振、恶心呕吐、口腔尿臭味 |
| **心血管系统** | 高血压、心力衰竭、心包炎；**心力衰竭是主要死因** |
| **血液系统** | **肾性贫血**（EPO↓）、出血倾向（血小板功能↓） |
| **呼吸系统** | Kussmaul呼吸（酸中毒）、尿毒症肺炎 |
| **神经系统** | 尿毒症脑病（淡漠→迟钝→昏迷）、周围神经病变 |
| **皮肤** | 尿素霜沉着→皮肤**瘙痒**、干燥、色素沉着 |
| **骨骼系统** | **肾性骨病**：低钙高磷→继发性甲旁亢→骨软化/骨质疏松/纤维性骨炎 |
| **水电酸碱** | **高钾**、高磷、低钙、高镁、代谢性酸中毒 |
| **内分泌** | EPO↓、活性VitD₃↓ |

### 🔑 名词解释
- **慢性肾衰竭**：各种慢性肾脏病进行性进展，肾单位进行性破坏，导致肾功能不可逆性丧失，以代谢产物潴留、水电解质酸碱紊乱及全身各系统受累为特征的临床综合征
- **尿毒症**：CRF的终末期，GFR<10ml/min，出现全身各系统严重中毒表现
- **肾性贫血**：CRF时因EPO生成减少导致的贫血
- **肾性骨病**：CRF时因低钙高磷、活性VitD₃↓、继发性甲旁亢导致的骨骼病变
- **蜡样管型**：CRF特征性尿液改变，由肾小管内变性坏死细胞和蛋白质形成

### 💊 治疗要点
1. **原发病治疗**+延缓肾衰竭进展
2. **饮食治疗**：优质低蛋白+高热量+低磷
3. **纠正水电酸碱紊乱**：补钙降磷（碳酸钙、醋酸钙）；降钾（限制+排钾利尿+透析）；纠酸（碳酸氢钠）
4. **肾性贫血**：重组人促红细胞生成素（rHuEPO）+铁剂
5. **肾性骨病**：活性VitD₃（骨化三醇）+ 磷结合剂
6. **替代治疗**：血液透析/腹膜透析/**肾移植**（最有效手段）

### 🏥 护理诊断
1. **体液过多**——与肾小球滤过率↓、水钠潴留有关
2. **营养失调：低于机体需要量**——与厌食、限制蛋白摄入有关
3. **活动无耐力**——与贫血、心功能不全有关
4. **有皮肤完整性受损的危险**——与水肿、尿素霜刺激所致瘙痒有关
5. **潜在并发症**：高钾血症、心力衰竭、感染

### 🤲 护理措施

**饮食护理（核心🔥）：**
- 优质低蛋白（0.6~0.8g/kg/d，以鱼、蛋、瘦肉等动物蛋白为主）
- 高热量（30~35kcal/kg/d，以碳水化合物和脂肪提供）
- 低磷（限制奶制品、坚果、动物内脏）
- 限制水钠钾：
  - 限盐<3g/d（高血压水肿时）
  - 限水=前日出量+500ml
  - 高钾食物：（香蕉、橘子、白菜、萝卜、土豆、红枣）→避免

**病情观察：**
- 每日测体重、记录出入量
- 监测血电解质（尤其血钾）+心电图（高钾→T波高尖）
- 观察有无恶心、头痛、意识改变

**安全护理：**
- **禁用库存血**（库存血含钾高）
- 动静脉瘘侧肢体：**禁止测血压、抽血、静脉输液、提重物**
- 皮肤瘙痒：温水擦洗、保持清洁、剪短指甲、避免搔抓

**用药护理：**
- 促红素：皮下注射，注意高血压副作用
- 碳酸钙：餐中嚼服（降磷效果最好）
- 纠正酸中毒时→同时补钙（防低钙抽搐）

## 八、血液净化治疗的护理

### 🔬 血液透析（HD）
- **原理**：弥散+对流+超滤，清除代谢废物和多余水分
- **血管通路**：动静脉内瘘（最常用）或中心静脉导管
- **最常见并发症**：**低血压**（超滤过多过快）
- **失衡综合征**（首次透析）：脑水肿→头痛、恶心、呕吐、肌肉痉挛
- **首次透析**原则：时间短（2~3h）、超滤少、血流量低

### 🩺 腹膜透析（PD）
- **原理**：腹膜为半透膜，利用浓度梯度进行弥散和渗透
- **最常见并发症**：**腹膜炎**（引流液混浊、腹痛、发热）
- **优势**：可居家操作、对心血管影响小

### 🔑 名词解释
- **血液透析**：利用半透膜原理，通过弥散、对流和超滤清除体内代谢废物和多余水分的血液净化方法
- **失衡综合征**：血液透析过程中或透析后不久出现的以神经系统症状为主的综合征，因血中溶质被快速清除后脑组织渗透压相对增高→脑水肿
- **动静脉内瘘**：将患者动脉与邻近静脉通过手术吻合建立的永久性血管通路

### 🤲 护理措施
1. **动静脉瘘护理**：瘘侧肢体**禁测血压、禁抽血、禁输液、禁提重物**；保持清洁；每日触摸震颤感/听血管杂音（消失=血栓形成→立即报告）
2. **透析前**：测体重、血压、脉搏；了解体重增长情况
3. **透析中**：监测生命体征q30~60min；观察低血压、头痛、肌肉痉挛
4. **透析后**：测体重；压迫穿刺点15~30min；防止体位性低血压
5. **腹透护理**：严格无菌操作；观察引流液颜色和清亮度；混浊=腹膜炎→送检+抗生素


# 第六章 血液系统疾病病人的护理

## 一、🔥 缺铁性贫血（IDA）

### 🔬 发病机制
- 铁摄入不足（婴幼儿辅食添加晚）、铁吸收障碍（胃切除后）、铁丢失过多（**慢性失血**：月经过多、消化道出血→最常见原因）
- 体内贮存铁耗尽→红细胞内血红蛋白合成↓→红细胞体积小、血红蛋白含量低→**小细胞低色素性贫血**

### 🩺 临床表现
1. **一般贫血表现**：面色苍白、乏力、头晕、心悸、活动后气促
2. **组织缺铁表现**（特征性）：
   - **反甲**（匙状甲）——最具特征性
   - 异食癖（嗜食泥土、冰块、淀粉）
   - 皮肤干燥、毛发干枯脱落
   - 口角炎、舌炎
   - 吞咽困难（Plummer-Vinson 综合征）

### 🔑 名词解释
- **缺铁性贫血**：体内贮存铁缺乏，使血红蛋白合成减少，导致红细胞体积变小、血红蛋白含量降低的小细胞低色素性贫血
- **反甲**：指甲变薄、变脆、中间凹陷呈匙状，为缺铁性贫血组织缺铁的典型体征
- **异食癖**：缺铁性贫血患者嗜食非食物性物质（泥土、冰块、墙皮等）的异常行为

### 💊 治疗要点
1. **病因治疗**：查明并去除缺铁原因（最重要！）
2. **口服铁剂**：硫酸亚铁/琥珀酸亚铁
3. **注射铁剂**：口服不耐受或吸收障碍者
4. **输血**：重度贫血时输浓缩红细胞

### 🏥 护理诊断
1. **活动无耐力**——与贫血致组织缺氧有关
2. **营养失调：低于机体需要量**——与铁摄入不足或丢失过多有关
3. **知识缺乏**——缺乏铁剂治疗知识

### 🤲 护理措施

**饮食护理：**
- 多食含铁丰富食物：动物肝脏、瘦肉、蛋黄、动物血、黑木耳、菠菜
- 促进铁吸收：**维生素C**（柑橘、番茄）
- 避免影响铁吸收：茶、咖啡、牛奶、蛋类、植物纤维

**口服铁剂护理（必考🔥）：**
1. **饭后或餐中服用**→减少胃肠道刺激
2. 从小剂量开始逐渐加量
3. 与**维生素C**同服→促进吸收（酸性环境）
4. **忌同服**：茶（鞣酸）、咖啡、牛奶、抗酸药、H₂受体拮抗剂
5. **大便变黑**→正常现象（铁与肠道H₂S结合为硫化铁），提前告知患者
6. **液体铁剂用吸管**→避免牙齿染色
7. **疗效观察**：网织红细胞3~5天↑，1周达高峰；血红蛋白2周后↑
8. 血红蛋白正常后**继续服3~6个月**→补足贮存铁

**注射铁剂护理：**
- 深部肌内注射（采用**Z型注射法**）
- 首次剂量宜小→观察过敏反应
- 备好肾上腺素

## 二、再生障碍性贫血（AA）

### 🔬 发病机制
- **骨髓造血干细胞数量减少和/或功能异常**（"种子"学说）
- 造血微环境缺陷（"土壤"学说）
- 免疫异常（"虫子"学说）：T细胞攻击造血干/祖细胞
- 病因：**氯霉素**（最常见药物）、苯、放射线、肝炎病毒

### 🩺 临床表现
**三联征：贫血+出血+感染**
- 贫血：进行性加重
- 出血：皮肤瘀斑、鼻出血、牙龈出血、月经过多；颅内出血（主要死因）
- 感染：呼吸道感染最常见
- **肝脾淋巴结一般不肿大**（与白血病鉴别！）

### 🔑 名词解释
- **再生障碍性贫血**：由多种原因导致骨髓造血功能衰竭，以外周血全血细胞减少（贫血、出血、感染）为特征的一组综合征
- **全血细胞减少**：外周血红细胞、白细胞、血小板均低于正常

### 💊 治疗要点
1. **雄激素（首选）**：丙酸睾酮/司坦唑醇→刺激肾脏产生EPO+直接刺激骨髓造血
2. **免疫抑制剂**：环孢素、抗胸腺细胞球蛋白（ATG）
3. **造血干细胞移植**：重型AA最根本的治疗（40岁以下首选）
4. **支持治疗**：输血、抗感染、止血

### 🏥 护理诊断
1. **活动无耐力**——与贫血有关
2. **有感染的危险**——与粒细胞减少有关
3. **有出血的危险**——与血小板减少有关
4. **潜在并发症**：颅内出血

### 🤲 护理措施
1. **预防感染**：病房消毒、限制探视、口腔/皮肤护理
2. **预防出血**：软毛牙刷、避免磕碰、尽量减少肌内注射、拔针后延长按压
3. **雄激素护理**：深部缓慢分层肌注（丙酸睾酮为油剂）；观察肝功能损害、男性化副作用（痤疮、多毛、闭经）
4. **心理护理**：长期疗程（3~6月见效）、鼓励坚持治疗

## 三、🔥 急性白血病（AL）

### 🔬 发病机制
- 造血干/祖细胞在分化某一阶段发生**恶性克隆性增殖**→骨髓中原始/幼稚细胞大量积聚（≥20%）→抑制正常造血
- 浸润器官组织（肝、脾、淋巴结、中枢神经系统）
- 病因：病毒（HTLV-1）、电离辐射、苯及衍生物、化疗药物、遗传

### 🩺 临床表现（四大表现）
1. **发热**：最常见症状之一，主要是**感染**所致（成熟粒细胞缺乏）
2. **出血**：皮肤瘀斑、牙龈出血、鼻出血；**颅内出血**是主要死因
3. **贫血**：进行性加重，与正常红细胞生成受抑制有关
4. **浸润表现**：
   - 肝、脾、淋巴结肿大
   - **胸骨下段压痛**（对诊断最有提示意义）
   - 骨关节疼痛
   - 牙龈增生（M4/M5型）
   - 绿色瘤（眼眶部位粒细胞肉瘤）

**中枢神经系统白血病（CNSL）：**
- 化疗药物难以通过血脑屏障→白血病细胞"庇护所"
- 表现：头痛、呕吐、颈项强直、视乳头水肿
- 防治：**甲氨蝶呤（MTX）鞘内注射**

### 🔑 名词解释
- **急性白血病**：造血干/祖细胞恶性克隆性增殖，骨髓中原始及幼稚细胞≥20%，抑制正常造血，浸润器官组织的恶性血液病
- **中枢神经系统白血病（CNSL）**：白血病细胞浸润脑膜或脑实质所致，因化疗药不易透过血脑屏障而成为复发根源
- **完全缓解（CR）**：化疗后白血病症状和体征消失，外周血三系正常，骨髓原始细胞<5%

### 💊 治疗要点
1. **化疗（核心）**：
   - ALL→**VP方案**（长春新碱+泼尼松）
   - AML→**DA方案**（柔红霉素+阿糖胞苷）
   - 全反式维A酸→M3型（急性早幼粒细胞白血病）
2. **CNSL防治**：MTX鞘内注射（化疗同时进行）
3. **造血干细胞移植**：根治手段
4. **支持治疗**：抗感染、输血、止血

### 🏥 护理诊断
1. **有感染的危险**——与成熟粒细胞减少有关
2. **有出血的危险**——与血小板减少有关
3. **活动无耐力**——与贫血、化疗反应有关
4. **口腔黏膜改变**——与化疗药物损伤有关
5. **潜在并发症**：颅内出血、尿酸性肾病

### 🤲 护理措施

**化疗护理（必考🔥）：**
1. **骨髓抑制期护理**：化疗后7~14天为最低谷→**保护性隔离**（WBC<1×10⁹/L或中性粒<0.5×10⁹/L）
2. **预防尿酸性肾病**：化疗期间多饮水>2000ml/d+**口服别嘌醇**+碱化尿液（碳酸氢钠）→尿量>2000ml/d
3. **药物外渗处理**：立即停药→保留针头接注射器回抽→拔针→局部冷敷/普鲁卡因封闭→抬高患肢
4. **柔红霉素**→心脏毒性（查ECG）| **环磷酰胺**→出血性膀胱炎（多饮水+美司钠预防）| **长春新碱**→末梢神经炎 | **甲氨蝶呤**→口腔黏膜炎
5. **鞘内注射后**：去枕平卧4~6小时

**预防感染：**
- 口腔护理：软毛牙刷+碳酸氢钠/氯己定漱口
- 肛周护理：便后清洗+1:5000高锰酸钾坐浴

**病情观察：**
- 出血：有无头痛呕吐（颅内出血）、皮肤瘀斑+牙龈+鼻出血
- 血小板<20×10⁹/L→**绝对卧床**；输注血小板

## 四、特发性血小板减少性紫癜（ITP）

### 🔬 发病机制
- **免疫因素**：体内产生抗血小板抗体（PAIgG）→血小板在脾脏被单核-巨噬系统破坏→外周血小板↓
- 骨髓巨核细胞代偿性增生，但成熟障碍

### 🩺 临床表现
- **皮肤黏膜出血**：瘀点、瘀斑（最早最常见）
- 鼻出血、牙龈出血、月经过多
- 内脏出血少见；**颅内出血**最危险
- 肝脾淋巴结不肿大

### 💊 治疗要点
1. **糖皮质激素**（首选）：泼尼松
2. **脾切除**：激素无效6个月以上或需大剂量维持者（有效率70%）
3. **免疫抑制剂**：长春新碱、环磷酰胺
4. **急症处理**：血小板<20×10⁹/L+严重出血→输注血小板+大剂量甲泼尼龙

### 🤲 护理措施
- 血小板<20×10⁹/L→绝对卧床
- 避免磕碰、用软毛牙刷、禁止挖鼻
- 减少肌内注射，注射后按压>5分钟

## 五、DIC（弥散性血管内凝血）

### 🔬 发病机制
- 各种致病因素→广泛微血管内**微血栓形成**→消耗大量凝血因子和血小板→**继发性纤溶亢进**
- 四大表现：**出血+休克+栓塞+溶血**
- 诊断标志：**D-二聚体↑**（最敏感）

### 🔑 名词解释
- **DIC**：多种致病因素导致广泛微血管内血栓形成，消耗大量凝血因子和血小板，引起以出血、休克、栓塞、溶血为特征的临床综合征

### 💊 治疗要点
- 核心：**治疗原发病+去除诱因**
- 抗凝：肝素（高凝期用）
- 补充凝血因子：新鲜冰冻血浆、血小板
- 抗纤溶：氨甲苯酸（纤溶亢进期谨慎使用）

# 第七章 内分泌与代谢疾病病人的护理

## 一、🔥 甲状腺功能亢进症（甲亢/Graves病）

### 🔬 发病机制
- **自身免疫性疾病**：产生TSH受体抗体（TRAb）→持续刺激甲状腺滤泡上皮→T3/T4分泌↑↑
- 遗传因素+环境诱因（感染、精神创伤）
- 最常见病理类型：**弥漫性毒性甲状腺肿（Graves病）**

### 🩺 临床表现
**高代谢综合征：** 怕热多汗+消瘦+多食易饥+乏力

**三大特征性表现：**
1. **甲状腺肿大**：弥漫性、对称、质软、可闻血管杂音
2. **突眼**：单纯性（交感兴奋）或浸润性（球后组织增生）
3. 胫前黏液性水肿

**各系统表现：**
- 神经精神：易激动、急躁、手震颤（细颤）
- 心血管：心动过速（静息仍快）、脉压增大、房颤
- 消化：多食易饥、大便次数增多
- 生殖：月经减少/闭经

### 🔑 名词解释
- **甲状腺功能亢进症**：甲状腺腺体产生过多甲状腺激素引起的一组临床综合征，以高代谢、甲状腺肿大和突眼为主要表现
- **甲状腺危象**：甲亢病情急剧恶化，出现高热>39℃、心率>140次/分、烦躁谵妄、恶心呕吐腹泻的致命危重状态
- **Graves病**：又称弥漫性毒性甲状腺肿，是甲亢最常见类型，属器官特异性自身免疫病

### 💊 治疗要点
| 方法 | 适应证 | 注意 |
|------|--------|------|
| 抗甲状腺药(PTU/MMI) | 轻中度、年轻 | **最严重副作用：粒细胞缺乏**（WBC<3x10^9/L停药） |
| 放射性131I | 25岁以上 | 主要并发症：永久性甲减 |
| 手术 | 中重度、压迫 | 可能损伤喉返神经/甲状旁腺 |

### 🏥 护理诊断
1. **营养失调：低于机体需要量**——与代谢率增高有关
2. **活动无耐力**——与高代谢有关
3. **有组织完整性受损的危险**——与突眼有关
4. **潜在并发症：甲状腺危象**

### 🤲 护理措施
**饮食护理：**
- 高热量+高蛋白+高维生素
- **禁食含碘食物**：海带、紫菜、海鲜
- 忌浓茶咖啡；多饮水2000~3000ml/d

**突眼护理：**
- 外出戴深色眼镜（防光防尘）
- 睡眠时抬高头部+涂眼膏+盖纱布（眼睑闭合不全时）
- 限水钠（减轻球后水肿）

**用药护理：**
- 服药前后**定期查血常规**（WBC<3x10^9/L或中性粒<1.5x10^9/L停药）
- 观察咽痛、发热（粒细胞缺乏早期表现）
- 总疗程1.5~2年，不可擅自停药

**甲状腺危象急救（🔥）：**
- 诱因：感染、手术、131I治疗后、应激
- 表现：高热>39℃+心率>140+烦躁谵妄+呕吐腹泻
- 处理：①PTU（抑制合成）→②碘剂（PTU后1h用，抑制释放）→③心得安（减慢心率）→④氢化可的松→⑤物理降温
- ⚠️ 禁用阿司匹林（使游离T4升高）

## 二、🔥🔥 糖尿病

### 🔬 发病机制
- **1型**：自身免疫破坏β细胞→**胰岛素绝对缺乏**→必须外源胰岛素
- **2型**：**胰岛素抵抗**（为主）+分泌相对不足
- 长期高血糖→AGEs形成→微血管病变（视网膜/肾脏）+大血管病变（心脑血管）

### 🩺 临床表现
**典型"三多一少"：** 多尿+多饮+多食+体重下降

| | 1型 | 2型 |
|------|------|------|
| 年龄 | 青少年 | >40岁中老年 |
| 体型 | 多消瘦 | 多肥胖 |
| 三多一少 | 明显 | 不明显 |
| 酮症倾向 | 易 | 不易 |
| 治疗 | **必须胰岛素** | 口服药±胰岛素 |

### 🔑 名词解释
- **糖尿病**：由遗传和环境因素共同作用引起的以慢性高血糖为特征的代谢疾病群
- **DKA**：糖尿病酮症酸中毒，因胰岛素严重缺乏→脂肪分解↑→酮体生成↑→代谢性酸中毒+脱水
- **Kussmaul呼吸**：DKA时深大呼吸，为机体代偿代谢性酸中毒
- **Somogyi效应**：夜间低血糖后清晨反跳性高血糖→**减少**睡前胰岛素
- **黎明现象**：清晨升糖激素↑导致的高血糖→**增加**睡前胰岛素

### 💊 治疗要点
**饮食治疗是所有治疗的基础**

| 口服降糖药 | 代表 | 作用 | 注意 |
|------|------|------|------|
| 双胍类 | **二甲双胍** | 胰岛素增敏 | 2型首选，不引起低血糖 |
| 磺脲类 | 格列本脲 | 促进分泌 | 餐前30min；副作用：低血糖 |
| α-糖苷酶抑制剂 | 阿卡波糖 | 延缓吸收 | **与第一口饭同服** |

### 🏥 护理诊断
1. **营养失调**——与胰岛素分泌/作用缺陷有关
2. **有感染的危险**——与高血糖、免疫力下降有关
3. **知识缺乏**——缺乏糖尿病自我管理知识
4. **潜在并发症：DKA、低血糖、糖尿病足**

### 🤲 护理措施

**胰岛素护理（必考🔥）：**
1. 未开封2~8℃；开封后室温≤25℃可用28天
2. 注射部位轮换：腹部（吸收最快）→上臂→大腿→臀部
3. **混合顺序：先抽短效（澄清）→再抽长效（混浊）**
4. 最常见不良反应=低血糖→清醒者口服糖水；昏迷者静推50%GS

**DKA急救（🔥案例分析）：**
1. 补液：首选**生理盐水**，先快后慢
2. 胰岛素：小剂量持续静滴（0.1U/kg/h）
3. 补钾：见尿补钾（尿量>40ml/h）
4. 补碱：pH<7.1时补碳酸氢钠
5. 监测：q1~2h测血糖

**糖尿病足护理：**
- 每日检查足部；温水<37℃洗脚；不赤脚；趾甲平剪

# 第八章 风湿性疾病病人的护理

---

## 一、🔥 系统性红斑狼疮（SLE）

### 🔬 发病机制
- 遗传+环境（**紫外线**、感染、药物）→免疫耐受破坏→B细胞过度活化→大量**自身抗体**（以ANA为代表）
- 抗原抗体复合物沉积→**血管炎**（基本病理）
- 雌激素↑→好发育龄女性（女:男=9:1）

### 🩺 临床表现（多系统）
| 系统 | 表现 |
|------|------|
| 皮肤 | **蝶形红斑**（特征性）、光过敏、口腔痛性溃疡 |
| 关节 | 对称性多关节痛，**极少畸形**（区别于RA） |
| 肾脏 | **100%受累**→慢性肾衰竭（主要死因） |
| 心血管 | 心包炎最常见 |
| 神经 | 神经精神性狼疮→预后差 |
| 血液 | 贫血、WBC↓、PLT↓ |

### 🔑 名词解释
- **SLE**：多系统损害的慢性系统性自身免疫病，以ANA为代表的自身抗体阳性，基本病理改变为血管炎
- **蝶形红斑**：SLE特征性面颊部蝶形分布的水肿性红斑
- **狼疮性肾炎**：SLE累及肾脏，几乎100%患者有肾脏病理改变

### 💊 实验室检查（必考🔥）
| 检查 | 意义 |
|------|------|
| **ANA** | 敏感度最高（>95%），**首选筛查** |
| **抗dsDNA抗体** | 与**疾病活动性成正比** |
| **抗Sm抗体** | **标记性抗体**（特异性99%） |
| 补体C3/C4 | 活动期↓ |

### 💊 治疗要点
1. **糖皮质激素**（首选）：泼尼松，起始足量、缓慢减量
2. 免疫抑制剂：**环磷酰胺**（重型/狼疮性肾炎）
3. 羟氯喹（皮肤+关节症状，需查眼底）

### 🏥 护理诊断
1. **皮肤完整性受损**——与血管炎及药物有关
2. **疼痛**——与自身免疫反应有关
3. **身体意象紊乱**——与面容毁损有关
4. **潜在并发症：慢性肾衰竭**

### 🤲 护理措施
**皮肤护理（🔥必考）：**
1. **严格避光**：长袖长裤+宽边帽+防晒霜SPF>30
2. 病室挂厚窗帘；紫外线消毒时回避
3. **温清水洗脸，禁用碱性肥皂**、化妆品、染发烫发
4. **饮食禁忌：芹菜、香菜、无花果**（含补骨脂素）

**用药护理：**
- 激素：不可骤停；观察感染/高血糖/高血压/骨质疏松
- 环磷酰胺：**多饮水**防出血性膀胱炎；定期血象+肝肾功

**生育指导：** 缓解期+**停免疫抑制剂半年以上**方可妊娠

## 二、🔥 类风湿关节炎（RA）

### 🔬 发病机制
- 遗传+环境→**免疫紊乱**→RF与IgG形成免疫复合物→沉积滑膜→**滑膜炎**（基本病理）→血管翳→软骨/骨质破坏→畸形

### 🩺 临床表现
| 特征 | 内容 |
|------|------|
| 最早症状 | **关节痛** |
| 特征体征 | 关节**梭形肿胀** |
| 关节 | **对称性四肢小关节**（腕/掌指/近端指间） |
| 晨僵 | **≥1小时**，与滑膜炎活动性成正比 |
| 晚期畸形 | **尺侧偏向**+"天鹅颈"+"纽扣花" |
| 关节外 | **类风湿结节**（最具特异性，表示活动） |

### 🔑 名词解释
- **类风湿关节炎**：以慢性、对称性、周围性多关节炎为特征的自身免疫病，基本病理为滑膜炎
- **晨僵**：早晨起床后关节僵硬/胶着感，活动后减轻，≥1h有意义
- **类风湿结节**：RA最具特征性关节外表现，出现在关节隆突受压处

### 💊 治疗要点
1. NSAIDs：对症止痛
2. **DMARDs（甲氨蝶呤首选）**：延缓关节破坏，应早期使用
3. 糖皮质激素：急性重症时短期桥接
4. 生物制剂：TNF-α拮抗剂

### 🏥 护理诊断
1. **疼痛**——与关节炎性反应有关
2. **有废用综合征的危险**——与关节疼痛、僵硬、畸形有关
3. **自理缺陷**——与关节肿痛畸形有关

### 🤲 护理措施
**体位护理（🔥必考）：**
- 活动期卧床，保持关节**功能位**
- 膝下**勿垫高枕**；足底放**护足板**防垂足
- 不宜长时间屈曲姿势

**晨僵护理：**
- 夜间关节**保暖**；晨起温水浸泡15~20min/热敷后活动

**功能锻炼：**
- 缓解期渐进活动；以不引起剧痛为度

# 第九章 神经系统疾病病人的护理

---

## 一、🔥🔥 脑血管疾病

### 🔬 发病机制
- **脑出血**：高血压→小动脉玻璃样变→微动脉瘤→血压骤升时破裂→血肿→颅内压↑→脑疝（最常部位：**内囊**）
- **脑梗死**：动脉粥样硬化→管腔狭窄→血栓形成→缺血坏死；常在**安静/睡眠中**发病

### 🩺 临床表现
| | 脑出血 | 脑梗死 |
|------|--------|------|
| 病因 | **高血压** | 动脉粥样硬化 |
| 发病 | **活动/激动时**急骤 | **安静/睡眠中**缓慢 |
| 头痛呕吐 | 明显 | 轻/无 |
| 意识障碍 | 多见 | 少见 |
| CT | **高密度** | **低密度** |
| 脑脊液 | 血性 | 正常 |

### 🔑 名词解释
- **脑出血**：原发性非外伤性脑实质内出血，最常见原因为高血压
- **脑梗死**：脑部血液循环障碍、缺血缺氧导致的局限性脑组织缺血性坏死
- **三偏症**：对侧偏瘫+偏身感觉障碍+同向偏盲（内囊区病变）

### 💊 治疗要点
**脑出血：**
- 甘露醇降颅压+绝对卧床+控制血压+避免搬动
**脑梗死：**
- **溶栓rt-PA（4.5h内）**+抗血小板

### 🏥 护理诊断
1. **有受伤的危险**——与偏瘫/意识障碍有关
2. **潜在并发症：脑疝**

### 🤲 护理措施（🔥必考）
1. 绝对卧床4周+床头抬高15°~30°
2. **避免搬动**、用力排便、情绪激动
3. 甘露醇快速静滴
4. 观察脑疝先兆：剧烈头痛+喷射性呕吐+瞳孔不等大+意识加深

## 二、🔥 癫痫

### 🔬 发病机制
- 大脑神经元**异常同步放电**→短暂大脑功能障碍

### 🩺 临床表现
- **大发作**：意识丧失+全身抽搐（最危险）
- **癫痫持续状态**：发作≥30min或连续发作间歇意识未恢复→**急症**

### 💊 治疗
- 持续状态首选：**地西泮静推**
- 用药原则：长期规律、不可骤停

### 🤲 护理措施（🔥必考）
1. ✅ 侧卧位+保持呼吸道通畅+移开危险物
2. ❌ **禁止掐人中、塞物入口、按压肢体、灌水喂药**

- **特征**：静止性震颤（搓丸样）+肌强直+运动迟缓（面具脸）+慌张步态

## 四、重症肌无力

- **肌无力危象**：呼吸肌无力→呼吸衰竭
- **治疗**：新斯的明/溴吡斯的明
